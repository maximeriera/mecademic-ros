## Overview

This repository contains a ROS 2 workspace for the Mecademic Meca500 robot arm, including:

- Robot description (URDF/Xacro and meshes)
- ros2_control bringup
- Hardware interface plugin
- MoveIt 2 configuration

The current workspace layout is based on a standard colcon structure:

- `src/` contains source packages
- `build/`, `install/`, `log/` are generated by colcon

## Packages

### 1) meca500_description

Description package for visualization and robot model assets.

- Main content: URDF/Xacro, meshes, RViz resources
- Launch file: `view_robot.launch.py`
- Typical use: inspect robot model and joint states in RViz

### 2) meca500_hardware

ROS 2 hardware interface plugin for integration with ros2_control.

- Main content: system hardware implementation
- Typical use: backend for real robot control through controller manager

### 3) meca500_bringup

Bringup package for ros2_control and controllers.

- Main content: controller configuration and startup launch
- Launch file: `meca500.launch.py`
- Typical use: start controller manager, robot_state_publisher, and controller spawners

### 4) meca500_moveit_config

MoveIt 2 configuration package for planning and execution.

- Main content: SRDF, kinematics, OMPL planning, MoveIt controllers
- Launch file: `move_group.launch.py`
- Typical use: run MoveIt `move_group` and RViz planning interface

## Prerequisites

This workspace is configured for ROS 2 Humble.

Required tools and components:

- ROS 2 Humble desktop or equivalent packages
- `colcon` and `rosdep`
- MoveIt 2 (for `meca500_moveit_config`)

## Build

From the workspace root (`ros2_meca500_ws`):

```bash
rosdep install --from-paths src --ignore-src -r -y
colcon build --symlink-install
source install/setup.bash
```

## Run Examples

### Visualize robot description

```bash
source install/setup.bash
ros2 launch meca500_description view_robot.launch.py
```

### Start ros2_control bringup

```bash
source install/setup.bash
ros2 launch meca500_bringup meca500.launch.py
```

### Start MoveIt 2

```bash
source install/setup.bash
ros2 launch meca500_moveit_config move_group.launch.py
```

For full operation, run bringup and MoveIt in separate terminals after sourcing `install/setup.bash` in each terminal.
